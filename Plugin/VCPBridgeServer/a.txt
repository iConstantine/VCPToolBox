## Linux 命令调用规范

当前运行环境为 **Windows + PowerShell 7.x + uutils coreutils**。所有终端命令**默认使用 Linux 风格语法**。

### 必须这样做

- 使用 Linux 命令：`ls`, `cat`, `rm`, `cp`, `mv`, `grep`, `find`, `echo`, `pwd`, `mkdir`, `touch`, `head`, `tail`, `wc`, `sort`, `uniq`, `diff`, `date`, `df`, `du`, `chmod`, `chown`
- 管道用 `|`，命令串联用 `&&` / `||`
- 路径用正斜杠 `/`，如 `C:/Program Files` 或 `/c/Projects`
- 搜索用 `grep`，不要用 `Select-String`
- 查找文件用 `find`，不要用 `Get-ChildItem -Recurse`
- 列表用 `ls -la`，不要用 `Get-ChildItem`
- 流处理：`cat file | grep "x" | sort | uniq -c`

### 禁止这样做

- 禁止使用 PowerShell cmdlet 替代 coreutils：`Get-ChildItem`, `Select-String`, `Where-Object`, `ForEach-Object`, `Write-Host`
- 禁止使用 PowerShell 别名语法：`$env:VAR`, `$null -eq`, `?` 三元, `??`
- 禁止用反斜杠 `\` 作路径分隔符
- 禁止用 `;` 串联命令
- 禁止用 PowerShell 字符串转义规则写正则
- 需要 PowerShell 模块（如 `Invoke-RestMethod` 做复杂 HTTP 请求时）
